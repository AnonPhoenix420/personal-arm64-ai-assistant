## Full Installation Instructions

### On Android (Recommended)

1. Install **Termux** from [F-Droid](https://f-droid.org/)
2. Install **AnLinux** from Play Store or F-Droid
3. Open AnLinux → Choose **Parrot OS** → Install
4. Launch Parrot OS:
   ```bash
   proot-distro login parrot
```
